Scroll Fade

By Lintee

This article demonstrates how to use scroll fade.

How to use

Option 1: Download

Download scrollFade.js

Put this script at the bottom of the body element and add the "scroll-fade" class to all elements you want to fade

<script src="scrollFade.js"></script>

Option 2: CDN

Put this script at the bottom of the body element and add the "scroll-fade" class to all elements you want to fade

<script src="https://cdn.jsdelivr.net/gh/Lintee12/scroll-fade@latest/bin/js/scrollFade.js"></script>
